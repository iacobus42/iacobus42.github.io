SCOTUS and the 2020 Election

Looking forward at the 2020 election and back at 2016, one thing that is clear is the critical importance of the “present” in regards to determining the ideological make up of the Supreme Court of the United States (SCOTUS) over the next several decades.

Recently, in discussions with a friend, the question of how long it would take until the SCOTUS returned to a Democratic majority (at least 5 Democratic justices) if Trump wins or Trump loses re-election in 2020.

We both immediately thought about simulating possible courts to get an idea. To that end, I pulled the Social Security Administrations Actuarial Life Tables for 2015 (most recent year of data) to model morality among sitting and future SCOTUS justices. I then started writing some code to simulate possible worlds.

The Simulation

Everything here on assumes that you have tidyverse and survival packages loaded.

First, we need to know who the President is. I decided that I could reasonably simulate each election as a binomial outcome with the probability that Democrats win being, at base, 50%. There is an adjustable advantage to the GOP based on the Electoral College being slightly more favorable in gop_ec. I will assume that gop_ec is a 5 percentage point advantage.

Then there are two other features I wanted to include. Namely, Presidents running for re-election seem more likely to be re-elected. I incorporate this with the first_term flag and the advantage being first_term_advantage. I assumed this is 25 percentage points. There is then a penalty applied for candidates running for election after their party has held the White House for the two prior cycles. I called this the third_term_penalty (perhaps the name isn’t great because Presidents cannot have third terms). I assumed this would be around 35 percentage points. The function simulate_election() simulates a single election and is defined below.

simulate_election <- function(current_party, first_term, gop_ec, first_term_advantage, third_term_penalty) {
  if (current_party == "D" & first_term == FALSE) {
    p_dems <- 0.5 - gop_ec - third_term_penalty
  } 
  if (current_party == "D" & first_term == TRUE) {
    p_dems <- 0.5 - gop_ec + first_term_advantage
  } 
  if (current_party == "R" & first_term == FALSE) {
    p_dems <- 0.5 - gop_ec + third_term_penalty
  } 
  if (current_party == "R" & first_term == TRUE) {
    p_dems <- 0.5 - gop_ec - first_term_advantage
  }
  winner <- ifelse(rbinom(1, 1, p_dems) == 1, "D", "R")
  return(winner)
}

Then I need to simulate many elections and keep track of which political party holds the White House in which years. I defined generate_potus() to do that work for me. It accepts as arguments the current_party that holds the White House, whether this is a first_term or not, the start_year to simulate elections, the end_year to end the simulations, and the gop_ec, first_term_advantage, and third_term_penalty arguments as described for simulate_election().

generate_potus <- function(current_party, first_term, start_year, end_year, 
                           gop_ec = 0.05, first_term_advantage = 0.25, 
                           third_term_penalty = 0.35) {
  party_in_power <- vector("list", 25)
  i <- 1
  for (year in seq(start_year, end_year, 4)) {
    winner <- simulate_election(current_party, first_term, gop_ec, first_term_advantage, third_term_penalty)
    if (winner == current_party & first_term == TRUE) {
      first_term <- FALSE
    } else if (winner == current_party & first_term == FALSE) {
      first_term <- TRUE
    } else {
      first_term <- TRUE
    }
    current_party <- winner
    party_in_power[[i]] <- tibble(
      year = year,
      winner = winner,
      first_term = first_term
    )
    i <- i + 1
  }
  party_by_year <- party_in_power %>%
    bind_rows() %>%
    mutate(year = purrr::map(year, ~ seq(. + 1, . + 4))) %>%
    select(winner, year) %>%
    unnest() %>%
    rename(wh_party = winner)
  return(party_by_year)
}

I nexted define a function to simulate the court. Justices leave the court in one of two ways:

  1. Justices will serve terms until they die and are replaced by the President in office on Jan 21 in the year they die or
  2. Justices will, upon turning 86, retire after the election of the next same party President who replaces them on the court.

Replacement justices are selected from a uniform distribution of ages between 50 and 60. Half of selected replacement justices are female and half are male.

The function sim_courts() does this. It accepts the initial court, a vector of years_to_simulate, the life_table data, and the party in charge of the white_houses by year.

sim_courts <- function(court, years_to_simulate, life_table, white_houses) {
  courts <- vector("list", length(years_to_simulate))
  
  for (year in years_to_simulate) {
    # find prob of death
    prob_death <- court %>%
      inner_join(life_table,
                 by = c("age", "female"))
    # simulate deaths and replacements, +1 to age for retained justices
    court <- prob_death %>%
      mutate(death = rbinom(9, 1, p_death)) %>%
      mutate(year = year) %>%
      inner_join(white_houses, by = "year") %>%
      mutate(party = ifelse(death, wh_party, party),
             age = ifelse(death, sample(50:60, 1), age + 1),
             female = ifelse(death, rbinom(1, 1, 0.5) == 1, female)) %>%
      select(party, age, female)
    
    # allow retirements
    if (year %% 4 == 1) {
      court <- court %>%
        mutate(year = year) %>%
        inner_join(white_houses, by = "year") %>%
        mutate(retires = party == wh_party & age > 85) %>%
        mutate(age = ifelse(retires, sample(50:60, 1), age),
               female = ifelse(retires, rbinom(1, 1, 0.5) == 1, female)) %>%
        select(party, age, female)
    }
    
    courts[[year - min(years_to_simulate) + 1]] <- court
  }
  
  balance <- courts %>%
    enframe() %>%
    mutate(year = name + min(years_to_simulate) - 1) %>%
    unnest() %>%
    group_by(year) %>%
    summarize(
      n_republican_justices = sum(party == "R")
    ) %>%
    mutate(party_in_control = ifelse(n_republican_justices >= 5, "R", "D"))
  return(balance)
}

Must focus has been put on Ginsburg’s age and health, and rightfully so, but Clarence Thomas might be an underappericated weakness. At 71 years old, assuming he does not retire this year, there is a 13% chance of mortality during the next president. Assume a Democrat wins the White House, there is a 27% chance of Thomas’s death during the 8 years between 2021 and 2028.

The reelection of Trump in 2020 would extend conservative control of the SCOTUS for more than 13 years in 50% of the simulations. Beyond simply the duration, the Republican lean of the court is likely to increase. If Trump wins in 2020, 17.9% of simulations have a GOP 7-2 or larger majority on the court compared to 13.3% with a Trump loss in 2020.

Bonus: How Would Have Merrick Garland Changed the Court

Merrick Garland was nominated to the court in 2016 by Obama; however, the Senate, controlled by Mitch McConnell, never brought Garland up for a vote. As a result, he was not elevated to the court and Neil Gorsuch filled that seat in early 2017.

Using this simulator, how did the court change as a result of Garland not being confirmed? In this hypothetical, Trump won the election in 2016 and Kennedy was still on the court. I do not code in Kennedy’s decision to retire in 2018 but instead treat his mortality and retirement decisions as any other justice. This would have been the status quo in 2016 when the Garland nomination was blocked.

I initalize two courts:

  1. garland_court is the court makeup with Garland confirmed to the court
  2. mcconnell_court is the court makeup with Garland’s nomination held up and Gorsuch filling to that seat.

I measure the importance of having Garland on the court, as well as the outcome of the 2020 election, by computing the percentage of time in the simulations that the court has a 5-4 or greater liberal majority between 2020 and 2050. This allows us to compare the relative importance of the 2020 election and the blocking of Garland’s nomination.

Having Garland on the court would have increased the number of years between 2020 and 2050 with a Democratic majority on the court by 21.93% if Democrats win in 2020 and 30.11% if Republicans win in 2020.

In contrast, losing the 2020 election to Trump would reduce the expected percentage of years with Democratic control of SCOTUS by 18.85% - a change from being the majority 41.14% of the time to a barely present 22.29% of the time.

Failing to confirm Garland was a major blow and had a generational effect on American law. It was perhaps the single most impactful event on the SCOTUS composition for the next 30 years. That said, it shifted the court from likely having a solid and lasting liberal majority for most of the next 30 years to having just under half of the next 30 years. A Trump victory in November would essentially double the impact and place the Democrats in the minority for the vast majority of the next 30 years. Such an outcome would likely be fatal to future progressive causes.
